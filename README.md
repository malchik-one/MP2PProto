## MP2PProto - UDP протокол изначально создоваемый для P2P свзи 2-х PC.

### Для чего?
Изначальное преминение это P2P мессенджер. Но протокол создает очень гибким с возможностью применения в любых облостях. По этому у протокола нет четкой структуры использует идет по принципу TLV (Type-Length-Value).
Так же протокол не завянзан на точном обозначении операции из-за чего быстрым путем можно заменить значении опрации к пример "Send" с 5 на 2 или 100 до 255 из-за размера в 1 байт для метки операции.
