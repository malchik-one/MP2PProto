## MP2PProto - UDP протокол изначально создоваемый для P2P свзи 2-х PC.

### Для чего?
Изначальное приминение это P2P (Point to Point) мессенджер. Протокол создается очень гибким. 
По этому у протокола нет четкой структуры написание идет по принципу TLV (Type-Length-Value).
Так же протокол не завянзан на точном обозначении операции из-за чего быстрым путем можно заменить значении опрации к пример "Send" с 5 на 2 или 100 до 255 из-за размера в 1 байт для метки операции. Так же такая гибкость нужна для обхода DPI или же ТСПУ что тоже явялется частью протокола.
